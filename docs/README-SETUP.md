### Próximos Pasos

1. **Instalar dependencias**:
   ```bash
   npm install
   ```

2. **Iniciar con Docker**:
   ```bash
   docker-compose up -d
   ```

3. **Desarrollo local**:
   ```bash
   npm start
   ```

4. **Acceder a la aplicación**:
   - Frontend: http://localhost:3000
   - CouchDB: http://localhost:5984/_utils

### Comandos Útiles

```bash
# Compilar para producción
npm run build

# Ejecutar tests
npm test

# Ver logs de Docker
docker-compose logs -f

# Parar servicios
docker-compose down
```
